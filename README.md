# Cython

## Оптимизация вычислений с использованием Cython

Проект демонстрирует ускорение критически важных для производительности участков кода (расчет суммы квадратов элементов массива) путем компиляции Python-кода в машинные инструкции языка C с использованием Cython.

### Структура проекта для Cython

*   **`my_math.pyx`** — Исходный код на языке Cython с реализацией функции `sum_of_squares`. За счет статической типизации и отключения внутренних проверок Python цикл выполняется со скоростью чистого C.
*   **`setup.py`** — Скрипт сборки. Отвечает за трансляцию кода из файла `my_math.pyx` в файл на языке C (`my_math.c`) и его последующую компиляцию в бинарный модуль `my_math.cpython-313-x86_64-linux-gnu.so`.
*   **`Makefile`** — Декларативное описание команд терминала для автоматизации сборки, запуска и очистки проекта от временных файлов.



# Numba

## Оптимизация вычислений с использованием Numba

Проект демонстрирует ускорение критически важных для производительности участков кода (расчет суммы квадратов элементов массива) путем компиляции Python-кода с применением numba.

### Структура проекта для Numba

*   **`main.py`** — Исходный код на языке Python с реализацией функции `sum_of_squares` и декоратором @jit. За счет принудительной компиляции через декоратор @jit происходит ускорение работы, цикл выполняется со скоростью чистого C.
*   **`Makefile`** — Декларативное описание команд терминала для запуска проекта.
